## Summary

## Features

- [] Create a short URL;
  - [] Link the short URL with authenticated user;
- [] Redirect to the original URL;
  - [] Track the number of clicks on the short URL;
- [] Create user accounts;
- [] Authenticate users;
- [] List all short URLs created by a user;
- [] Delete a short URL;
- [] Update a short URL;
